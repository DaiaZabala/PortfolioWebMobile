Quickmart

Plataforma web destinada a la gestión de un supermercado, donde los usuarios pueden administrar productos, realizar ventas y gestionar clientes. La aplicación permite a los administradores agregar, editar y eliminar productos del inventario, realizar ventas y generar tickets de compra, todo mientras se actualiza el stock en tiempo real. Además, los datos de los clientes, incluidos su historial de compras, se gestionan de manera eficiente. El proyecto ha sido desarrollado por un equipo de trabajo, como proyecto final del curso de Desarrollador FullStack dictado por Talentos Digitales y TELCO, utilizando principalmente Laravel (PHP) para el backend, Blade para las plantillas del frontend y MySQL como sistema de base de datos.
